2D Cartesian Grapher

Enter a function and press Graph.

Use *, ^, sin(x), cos(x), exp(x), sqrt(x), pi, e

Input guide

Algebra: x^2 - 4*x + 1

Trig: sin(x) + cos(2*x)

Derivative: enter f(x), the app graphs f'(x).

Integral: shades signed area and graphs cumulative numeric integral.

Vector field: enter component functions such as P=-y, Q=x.

ODE: enter dy/dx = f(x,y); solved numerically using RK4.
